## 大一下课程

### 程序设计 2（C++）（冯欣）

&emsp;学习 C++面向对象编程，难度小于 C，可以类比学习遣词造句(程序设计 1)和学习句子的修辞手法(程序设计 2)。虽然难度稍小，但是会经常出现写半天代码编译一堆错误的情况，这个时候要冷静，一点一点找问题，实在有困难也可以和同学交流。<br>
&emsp;考核方式与 C 相同。<br><br>

### python（限选）

&emsp;学习 python 的基础语法，并介绍一些常见的库或者运用（如 numpy、网络爬虫）。python 和 C 的语法差别较大，但是总体上是比 C 更加方便的，我们学习的重点也是 python 的一些特有的语法，如列表、字典。<br>
&emsp;这里为大家整理了很多课件，可以作为参考资料。有了 C 语言的基础，通过查找资料和上手实践掌握 python 的基础语法是很自然的。<br>
&emsp;考核方式：2020 级为闭卷笔试（选择填空改错编程等），2021 级没返校是线上期中+小测验+大作业。<br><br>
