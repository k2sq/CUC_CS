大一下课程

程序设计 2（C++）（冯欣）

 学习 C++面向对象编程，难度小于 C，可以类比学习遣词造句(程序设计 1)和学习句子的修辞手法(程序设计 2)。虽然难度稍小，但是会经常出现写半天代码编译一堆错误的情况，这个时候要冷静，一点一点找问题，实在有困难也可以和同学交流。
 考核方式与 C 相同。

python（限选）

 学习 python 的基础语法，并介绍一些常见的库或者运用（如 numpy、网络爬虫）。python 和 C 的语法差别较大，但是总体上是比 C 更加方便的，我们学习的重点也是 python 的一些特有的语法，如列表、字典。
 这里为大家整理了很多课件，可以作为参考资料。有了 C 语言的基础，通过查找资料和上手实践掌握 python 的基础语法是很自然的。
 考核方式：2020 级为闭卷笔试（选择填空改错编程等），2021 级没返校是线上期中+小测验+大作业。

资料：链接：https://pan.baidu.com/s/1oQMB5BEnFLXdG7W8iiq2-Q 提取码：1234

逻辑与计算机设计基础（王晓宏）


(图为最新的第七版，之前用的是白中英第六版，具体用书请参考本学习的书目表)

 学习逻辑代数、各种组合逻辑电路，数学难度不大，实践性较强。2020 级在校是上课+实验（连电路）+闭卷考试（有复习 ppt），2021 级在家是视频课+软件模拟实验+大作业。
 2021 级大作业内容：链接：https://pan.baidu.com/s/1wruyQaD_o7cVWfxbeOvuSg 提取码：1234

高等数学上 & 概率论

 总体可以参考高数上   线性代数的模式，但是难度相对大一些，后期概念很抽象第一遍可能都听不懂。高数要重视平时作业的解法，即使不是独立完成的作业也要保证以后会做，概率论听不懂的话可以看一看辅导书和网课，比书上的概念更容易理解，同时作业的把握也很重要。

书主要用来看例题、写作业，电子版也够用。
